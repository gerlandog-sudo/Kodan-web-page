# Kodan Software · Product Facts
> Fecha de verificación: 2026-05-15

## 🛠️ Stack Tecnológico
- **Framework**: Vue 3 (Composition API).
- **Bundler**: Vite.
- **3D Engine**: Three.js + TresJS (@tresjs/core, @tresjs/cientos).
- **Animación**: GSAP + ScrollTrigger.
- **Styling**: Tailwind CSS (v4) + PostCSS.
- **CMS/Backend**: Sanity.io (@sanity/client).

## 🚀 Capacidades del Producto
- **Modernización de Legado**: Especialización en transformar sistemas antiguos.
- **Desarrollo a Medida**: Software de alto rendimiento.
- **Showcase 3D**: Visualización avanzada de componentes (CylinderShowCase).

## 📊 Especificaciones de Diseño
- **Dark Mode Native**: La aplicación no soporta Light Mode por diseño estratégico.
- **Resolución de Assets**: Hero assets en PNG/JPG de alta resolución.
- **Performance**: Optimización de hilos principales mediante carga diferida de texturas Three.js.
